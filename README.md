# hadoop入门map-reduce程序

hadoop 1.x.x版本，只需要一个jar:  
hadoop-core  

hadoop 2.x.x版本，需要引入4个jar:
hadoop-common  
hadoop-hdfs  
hadoop-mapreduce-client-core  
hadoop-client  
jdk.tools 一般也需要引入  

hadoop 3.x.x，看这本书：  
http://torlone.dia.uniroma3.it/bigdata/E1-Hadoop.pdf  
hadoop-common  
hadoop-mapreduce-client-core
